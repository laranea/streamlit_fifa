# 
### Steps to Run

Clone/Download- 
```bash
$ git clone https://github.com/rujual/streamlit_fifa.git
```

Install Streamlit Python library- 
```bash
$ pip install streamlit
```

Open terminal and Go into downloaded directory-

Run - 
```bash
$ streamlit run fifa.py
```
